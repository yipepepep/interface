Proxy

Configure how Windows connects to the internet using a proxy server.

Step 1: Access Proxy Settings

To access proxy settings, navigate to Settings > Network & Internet > Proxy from the Windows Start menu.

You can also access these settings by typing "proxy" in the Windows search box.

Step 2: Configure Automatic Proxy Detection

Automatically detect settings

Windows will try to detect proxy settings automatically.

Step 3: Configure Automatic Proxy Script

Use setup script

Use an automatic configuration script to configure proxy settings.

Step 4: Configure Manual Proxy Setup

Use a proxy server

Manually set up proxy settings.

Additional Information

When you configure a proxy server, Windows uses it for all network connections, including Microsoft Edge and other Windows apps.
Some applications may not use the Windows proxy settings. You might need to configure proxy settings separately for those applications.

# WINDOWS PROXY CONFIGURATION GUIDE This guide will help you set up proxy settings in Windows. ## METHOD 1: WINDOWS SETTINGS (RECOMMENDED) 1. Open Windows Settings: - Click on Start Menu - Click on the gear icon (Settings) - Or press Win+I on your keyboard 2. Navigate to Network & Internet section: - Click on "Network & Internet" - Select "Proxy" from the left sidebar 3. Configure your proxy: A) For Automatic Proxy Detection: - Toggle ON "Automatically detect settings" - Windows will try to detect network proxy settings automatically B) For Proxy Auto-Configuration Script: - Toggle ON "Use setup script" - Enter the address provided by your network administrator Example: http://proxy.example.com/proxy.pac - Click Save C) For Manual Proxy Setup: - Toggle ON "Use a proxy server" - Enter the proxy server address (e.g., proxy.example.com) - Enter the port number (e.g., 8080) - Check "Don't use proxy server for local addresses" if needed - Click Save 4. For Exceptions: - Under the Manual Proxy section, you can add addresses that should bypass the proxy - Use semicolons to separate multiple entries - Example: *.local;*.intranet;192.168.* ## METHOD 2: CONTROL PANEL 1. Open Control Panel: - Right-click Start Menu - Select "Control Panel" - Or type "Control Panel" in search 2. Navigate to Internet Properties: - Click "Network and Internet" - Click "Internet Options" 3. Access Proxy Settings: - Go to "Connections" tab - Click "LAN settings" button at the bottom 4. Configure proxy in the dialog box: - Similar options as in Windows Settings - Check "Use a proxy server for your LAN" - Enter Address and Port - Click OK when done ## IMPORTANT NOTES - Proxy settings apply system-wide to Windows and most applications - Some applications may ignore Windows proxy settings and require separate configuration - You may need to restart applications for proxy changes to take effect - Contact your network administrator if you don't know your proxy settings ## TESTING YOUR PROXY To test if your proxy is working: 1. Open a web browser 2. Try to access a website (e.g., www.google.com) 3. If it works, your proxy is configured correctly If you cannot connect to the internet: - Double-check your proxy address and port - Verify your proxy credentials if authentication is required - Check if your proxy server is accessible - Try temporarily disabling the proxy to see if that's the issue ## TROUBLESHOOTING If you experience issues: - Restart your computer after changing proxy settings - Check if your proxy server requires authentication - Try using automatic detection first - Consult your IT department or network administrator
